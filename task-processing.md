### FI-006: 入力補完・ショートカット

**説明:** ユーザーとして、入力を効率化するショートカットや補完がほしい。

**受け入れ基準:**

- [ ] ASCII入力からUnicode変換のリアルタイム候補表示（例: `->` 入力中に `→` を候補表示）
- [ ] ギリシャ文字のASCII名→Unicode変換候補（例: `phi` → `φ`）
- [ ] Tab/Enterで候補選択
- [ ] 量化子テンプレート挿入（`all` → `∀.` のスケルトン）
- [ ] Storybookストーリーで補完動作を確認
- [ ] 型チェック/lintが通る

### 設計メモ

- 補完ロジックはピュアロジックとして分離（テスタブルに）
- 補完候補のデータ: ASCII→Unicode マッピングテーブル
- UIはポップアップ候補リストとして FormulaInput/TermInput に統合
- キーボード操作: Tab/Enter で確定、Escape でキャンセル、矢印キーで候補移動
